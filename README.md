# delphi-ota
Laboratorio de testes para projetos Delphi - Open Tools API

Este plugin adiciona menus e recursos de design-time ao Delphi 10.1 Berlin.

- Menu customizado no IDE com ações OTA.
- Tela de configuração de telemetria disponível pelo menu.
- Build verificado com `msbuild` para `Debug|Win32` após a criação do formulário de telemetria.
